# Alura MIDI

Este é um projeto desenvolvido durante o curso da Alura sobre JavaScript, onde criamos um instrumento musical virtual (MIDI) que permite tocar sons de bateria usando o teclado do computador.

## Tecnologias Utilizadas

- **HTML5**: Estrutura da página web e elementos de interface.
- **CSS3**: Estilização da interface, incluindo layouts responsivos e animações.
- **JavaScript (ES6+)**: Lógica de programação para capturar eventos do teclado, reproduzir sons de áudio e interagir com o DOM.

## Como Usar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em um navegador web moderno (Chrome, Firefox, etc.).
3. Use as teclas do teclado para tocar os sons da bateria:
   - Tecla 'A': Tom 1
   - Tecla 'S': Tom 2
   - Tecla 'D': Tom 3
   - Tecla 'F': Tom 4
   - Tecla 'G': Surdo
   - Tecla 'H': Prato
   - Tecla 'J': Chimbal
   - Tecla 'K': Caixa
4. Cada tecla pressionada reproduzirá o som correspondente e poderá acionar animações visuais.

## Estrutura do Projeto

- `index.html`: Arquivo principal da página web.
- `main.js`: Script JavaScript com a lógica do instrumento MIDI.
- `css/estilos.css`: Folhas de estilo para a aparência da interface.
- `css/reset.css`: Reset de estilos para consistência entre navegadores.
- `images/`: Imagens utilizadas na interface (como ícones das teclas).
- `sounds/`: Arquivos de áudio MP3 dos sons da bateria.

## Funcionalidades

- Reprodução de sons de bateria via teclado.
- Interface visual responsiva.
- Animações ao pressionar as teclas.
- Suporte a múltiplas teclas simultâneas para ritmos complexos.

Este projeto demonstra conceitos fundamentais de JavaScript como manipulação do DOM, eventos e integração com elementos de áudio.
